# 🎮 AI-Generated RPG

Jeu de rôle (RPG) révolutionnaire avec intelligence artificielle générative créant des mondes, personnages et histoires infiniment uniques.

## ✨ Fonctionnalités principales

- **Mondes infinis** : Chaque partie est unique grâce à l'IA générative
- **Personnages dynamiques** : AI qui s'adapte à votre style de jeu
- **Histoire immersive** : Une narration qui évolue avec vos actions
- **Combat dynamique** : Système de combat tactique et stratégique
- **Progression de personnages** : Niveaux, compétences, équipements
- **Graphismes immersifs** : Design moderne et atmosphérique
- **Multiplateforme** : Compatible avec PC, mobile et web

## 🛠️ Stack technologique

- **Game Engine** : Custom engine en JavaScript/TypeScript
- **IA Générative** : GPT-4 pour créer du contenu unique
- **Graphismes** : Canvas API + WebGL
- **Audio** : Web Audio API pour des effets sonores immersifs
- **Multiplateforme** : Web, mobile, desktop

## 📦 Installation

```bash
# Clonez le dépôt
git clone https://github.com/crepin7/ai-rpg.git
cd ai-rpg

# Installez les dépendances
npm install

# Lancez le jeu
npm start
```
